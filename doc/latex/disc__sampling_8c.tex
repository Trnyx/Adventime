\doxysubsection{Description détaillée}
Génération de \char`\"{}\+Poisson Disc Sampling\char`\"{}. 

Poisson Disc Sampling est une technique permettant de sélectionner de manière aléatoire des points serrés de façon à ce qu\textquotesingle{}ils respectent une distance minimale Comme les points sont choisis de façon aléatoire, le résultat a un aspect plus organique


\begin{DoxyEnumerate}
\item Un premier point est généré aléatoirement et est stocké dans un tableau
\item Pour chaque nouveau points généré on regarde la distance le séparant de tous les autres points
\item Si ce dernier est trop loin ou trop proche alors on l\textquotesingle{}ignore et créer un nouveau point
\end{DoxyEnumerate}
