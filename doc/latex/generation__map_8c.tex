\doxysubsection{Description détaillée}
Génération d\textquotesingle{}une map. 

Voici l\textquotesingle{}algorithme de création d\textquotesingle{}une map \+:
\begin{DoxyEnumerate}
\item Pour chacuns des chunks de la map, un biome est sélectionné aléatoirement
\item Une fois tous les biomes choisis, on normalise la map NOMBRE\+\_\+\+DE\+\_\+\+NORMALISATION\+\_\+\+MAP fois
\item Pour chacuns des blocks d\textquotesingle{}un chunk, un tag est sélectionné aléatoirement, celui ci dépendant du biome du chunk dans lequel il se situe
\item Une fois tous les tags des blocks choisis, on normalise les blocks entre eux NOMBRE\+\_\+\+DE\+\_\+\+NORMALISATION\+\_\+\+CHUNK fois
\item A chaque normalisation d\textquotesingle{}un chunk on le lisse afin de supprimer (le plus possible) les éventuels blocks qui seraient rester seul lors de l\textquotesingle{}étape de la normalisation
\item Une fois tous les blocks normalisé, on place la végétation
\end{DoxyEnumerate}

Ce que nous appelons la \char`\"{}normalisation\char`\"{} est le fait \char`\"{}d\textquotesingle{}applatire\char`\"{} les chunks / les blocks entre eux chacun s\textquotesingle{}élève, s\textquotesingle{}abaisse ou ne bouge pas en fonction de ce qui l\textquotesingle{}entoure
